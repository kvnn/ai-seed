## Seed Invariants

- The frontend lives in `frontend/`.
- The Python package root lives in `backend/`.
- Domain-owned backend code goes in `backend/apps/<domain>/...`.
- Cross-cutting adapters go in `backend/services/...`.
- Database models stay centralized in `backend/models.py`.
- `docs/VISION.md` remains the primary source of truth for product intent.
